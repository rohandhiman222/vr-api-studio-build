## Install

**Linux**
```bash
sudo dpkg -i vr-api-studio-linux-amd64.deb
```

**Windows** — Run `vr-api-studio-windows-amd64-installer.exe`

**macOS** — Unzip `vr-api-studio-macos.zip` → drag to Applications
> First launch: right-click → Open (Gatekeeper bypass for unsigned app)

## Features

- **REST** — HTTP requests with full headers, body, auth, assertions
- **GraphQL** — Query editor with schema introspection
- **gRPC** — Protobuf-based RPC calls with streaming support
- **WebSocket** — Bidirectional real-time messaging
- **SSE** — Server-Sent Events live stream viewer
- **Mock Server** — Local mock endpoints for development
- **Collection Runner** — Run multiple requests in sequence
- **Webhook Catcher** — Receive and inspect incoming webhooks
- **Response Diff** — Compare two API responses side by side
- **AI Agent** — AI-powered API testing assistant
- **OpenAPI / Swagger** — Import and explore API specs
